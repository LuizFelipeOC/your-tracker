# Your Tracker 🚚

Hello and welcome in **YourTracker**, this app is experimental created after have  bad experiencies using the correios website braziliam. 
<br />
Here you have the two steps of searching for your packages, one of the steps is a simple search where you try to find by tracking code and the second step you can save the various codes and view the tracking at any time

### Techs

I decided use the libraries published of Flutterando (The larger Braziliam Community of Flutter), these being:
  * **Flutter Modular -> flutter pub get flutter_modular**
  * **Uno (Http Requestes) -> flutter pub get uno**

Others libaries usage are to save info in cache of device and mock values in units testes 

### API'S

This app use the Link&Track API to tracking the packages, the API is free, any people can use in personal apps. To use you need enter in contact with: 
  * **https://api.linketrack.com/api?utm_source=footer**
